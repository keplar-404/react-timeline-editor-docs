<Steps>
  <Step>
    ## Install the Package

    To get started, install `@keplar-404/react-timeline-editor` using npm:

    ```bash
    npm install @keplar-404/react-timeline-editor
    ```

  </Step>

  <Step>
    ## Render Your First Timeline

    Import the `Timeline` component and render it with some empty configuration.

    ```tsx
    import React from 'react';
    import { Timeline } from '@keplar-404/react-timeline-editor';

    export const TimelineEditor = () => {
      return (
        <Timeline
          editorData={[]}
          effects={{}}
        />
      );
    }
    ```

    <Callout type="info">
      While this example renders an empty timeline, it confirms that your installation was successful and the component mounts correctly.
    </Callout>

  </Step>

</Steps>
