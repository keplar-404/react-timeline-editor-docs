## Overview

`LoopZoneOverlay` renders a draggable repeat region overlay on top of `<Timeline>`. Requirements:

1. Mounted as a **sibling** to `<Timeline>` inside a `position: relative` wrapper
2. Given the **same geometry props** as `<Timeline>` (`scale`, `scaleWidth`, `startLeft`)
3. Kept in sync with the timeline's `scrollLeft` via the `onScroll` callback

## Props Reference

| Prop | Type | Required | Default | Description |
|:---|:---|:---|:---|:---|
| `scale` | `number` | ✅ | — | Must match `<Timeline>` `scale` prop |
| `scaleWidth` | `number` | ✅ | — | Must match `<Timeline>` `scaleWidth` prop |
| `startLeft` | `number` | ✅ | — | Must match `<Timeline>` `startLeft` prop |
| `scrollLeft` | `number` | ✅ | — | Current horizontal scroll offset |
| `loopStart` | `number` | ✅ | — | Loop region start time in seconds |
| `loopEnd` | `number` | ✅ | — | Loop region end time in seconds |
| `onLoopStartChange` | `(time: number) => void` | ✅ | — | Called when user drags the start handle |
| `onLoopEndChange` | `(time: number) => void` | ✅ | — | Called when user drags the end handle |
| `config` | `LoopZoneConfig` | — | `{}` | Visual configuration |
| `renderHandle` | `(side: 'start' \| 'end') => ReactNode` | — | — | Custom drag handle renderer |

## `LoopZoneConfig`

| Option | Type | Default | Description |
|:---|:---|:---|:---|
| `bandColor` | `string` | `'#3b82f6'` | CSS color for the loop region highlight |
| `bandOpacity` | `number` | `0.1` | Opacity of the highlight (0–1) |
| `showBoundaryLines` | `boolean` | `true` | Show dashed vertical lines at loop edges |

## Usage Example

```tsx
import React, { useState } from 'react';
import { Timeline, LoopZoneOverlay } from '@keplar-404/react-timeline-editor';

export const App = () => {
  const [loopStart, setLoopStart] = useState(2);
  const [loopEnd, setLoopEnd] = useState(8);
  const [scrollLeft, setScrollLeft] = useState(0);

  const scale = 1;
  const scaleWidth = 160;
  const startLeft = 20;

  return (
    <div style={{ position: 'relative' }}>
      <Timeline
        scale={scale}
        scaleWidth={scaleWidth}
        startLeft={startLeft}
        editorData={[]}
        effects={{}}
        onScroll={(p) => setScrollLeft(p.scrollLeft)}
      />

      <LoopZoneOverlay
        scale={scale}
        scaleWidth={scaleWidth}
        startLeft={startLeft}
        scrollLeft={scrollLeft}
        loopStart={loopStart}
        loopEnd={loopEnd}
        onLoopStartChange={setLoopStart}
        onLoopEndChange={setLoopEnd}
        config={{
          bandColor: '#10b981',
          bandOpacity: 0.12,
          showBoundaryLines: true,
        }}
      />
    </div>
  );
};
```

## Custom Handle Renderer

```tsx
<LoopZoneOverlay
  {/* ...other props... */}
  renderHandle={(side) => (
    <div className={`handle handle-${side}`}>
      {side === 'start' ? '⟨' : '⟩'}
    </div>
  )}
/>
```
