## Overview

`CutOverlay` provides a visual "blade" that lets users split action blocks at a time precise position.

Requirements:
1. Mounted as a **sibling** to `<Timeline>` inside a `position: relative` wrapper
2. Given the **same geometry props** as `<Timeline>`

## `<CutOverlay />` Props

| Prop | Type | Required | Default | Description |
|:---|:---|:---|:---|:---|
| `data` | `TimelineRow[]` | ✅ | — | Current timeline data |
| `scale` | `number` | ✅ | — | Must match `<Timeline>` `scale` |
| `scaleWidth` | `number` | ✅ | — | Must match `<Timeline>` `scaleWidth` |
| `scaleSplitCount` | `number` | ✅ | — | Must match `<Timeline>` `scaleSplitCount` |
| `startLeft` | `number` | ✅ | — | Must match `<Timeline>` `startLeft` |
| `rowHeight` | `number` | ✅ | — | Must match `<Timeline>` `rowHeight` |
| `editAreaTopOffset` | `number` | ✅ | — | Height of the time ruler (usually `32`) |
| `gridSnap` | `boolean` | — | `false` | Snap blade to grid intervals |
| `config` | `CutOverlayConfig` | — | `{}` | Visual and behavior config |
| `onModifierChange` | `(isActive: boolean) => void` | — | — | Called when keyboard modifier toggles |
| `onCut` | `(rowId: string, actionId: string, cutTime: number) => void` | ✅ | — | Called when user clicks to cut |

## `CutOverlayConfig`

| Option | Type | Default | Description |
|:---|:---|:---|:---|
| `bladeColor` | `string` | `'#3b82f6'` | Color of the blade line |
| `showPill` | `boolean` | `true` | Show floating time label pill |
| `formatPillLabel` | `(time: number) => string` | `"${time.toFixed(2)}s"` | Custom label formatter |
| `pillColor` | `string` | `'#1e3a8a'` | Label background color |
| `pillTextColor` | `string` | `'#ffffff'` | Label text color |
| `showBlockHighlight` | `boolean` | `true` | Highlight the hovered action block |
| `blockHighlightColor` | `string` | `'rgba(59,130,246,0.15)'` | Block highlight fill |
| `blockHighlightBorderColor` | `string` | `'rgba(59,130,246,0.4)'` | Block highlight border |
| `cursor` | `string` | `'crosshair'` | CSS cursor style |
| `keyboardModifier` | `string` | `--` | Key to hold to activate cut mode (e.g. `'c'`, `'Shift'`) |

## `splitActionInRow` Utility

Immutably splits a single action in a row at a given time. Returns a **new** `TimelineRow[]` array.

```typescript
function splitActionInRow(
  data: TimelineRow[],
  rowId: string,
  actionId: string,
  cutTime: number,
): TimelineRow[]
```

### Parameters

| Parameter | Type | Description |
|:---|:---|:---|
| `data` | `TimelineRow[]` | Current timeline data array |
| `rowId` | `string` | ID of the row containing the action |
| `actionId` | `string` | ID of the action to split |
| `cutTime` | `number` | Time in seconds at which to cut |

### Return Value

A new `TimelineRow[]` array with the target action replaced by two new actions:
- First: `[original.start, cutTime)`
- Second: `[cutTime, original.end)`

Both inherit the same `effectId` and `data` from the original.

### Example

```typescript
import { splitActionInRow } from '@keplar-404/react-timeline-editor';

const data: TimelineRow[] = [
  {
    id: 'row-1',
    actions: [{ id: 'action-1', start: 0, end: 10, effectId: 'effect0' }],
  },
];

const newData = splitActionInRow(data, 'row-1', 'action-1', 5);
// Result: two actions → [0, 5] and [5, 10]
// The original 'action-1' is replaced by two new actions with unique IDs
```
