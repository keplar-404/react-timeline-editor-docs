The `CutOverlay` provides a visual, interactive blade tool to split action blocks at a specific time cursor. This is a common feature in professional video and audio editors.

## Basic Usage

To implement a cutting feature, you mount the `<CutOverlay>` as a sibling to the `<Timeline>` inside a `position: relative` wrapper. When the user hovers over an action block, a precision blade appears.

```tsx
import React, { useState } from "react";
import {
  Timeline,
  CutOverlay,
  splitActionInRow,
} from "@keplar-404/react-timeline-editor";

export const EditorCutBlock = () => {
  const [data, setData] = useState(initialData);
  const [isCutMode, setIsCutMode] = useState(false);

  const handleCut = (rowId, actionId, cutTime) => {
    // We use the built-in splitActionInRow utility for immutable data updates
    const newData = splitActionInRow(data, rowId, actionId, cutTime);
    setData(newData);
  };

  return (
    <div style={{ position: "relative" }}>
      <button onClick={() => setIsCutMode(!isCutMode)}>
        {isCutMode ? "Disable Blade" : "Enable Blade (Hold C)"}
      </button>

      <Timeline editorData={data} onChange={setData} disableDrag={isCutMode} />

      <CutOverlay
        data={data}
        config={{ keyboardModifier: "c" }}
        onModifierChange={setIsCutMode}
        onCut={handleCut}
      />
    </div>
  );
};
```

## How It Works

1.  **Mounting**: The `CutOverlay` layer sits invisibly over the timeline area.
2.  **Activation**: You can toggle cut mode manually, or use the `keyboardModifier` (like `c` or `Shift`) to activate it on-the-fly.
3.  **Interaction**: When active, clicking on an action block will trigger `onCut`.
4.  **Utilities**: Use the `splitActionInRow` helper function to handle the logic of splitting one block into two while maintaining unique IDs and correct timing.

## Configuration

The `CutOverlay` accepts a `config` object for fine-grained visual control:

| Option               | Type      | Description                                       |
| :------------------- | :-------- | :------------------------------------------------ |
| `bladeColor`         | `string`  | Color of the vertical blade line.                 |
| `showPill`           | `boolean` | Show a label at the cursor indicating split time. |
| `showBlockHighlight` | `boolean` | Highlight the block currently under the blade.    |
| `keyboardModifier`   | `string`  | Key code (e.g., 'Shift', 'c') to hold to enable.  |

<Callout type="warning">
  Remember to set `disableDrag={true}` on the main `<Timeline />` when cut mode
  is active to prevent accidental block movements while splitting!
</Callout>
